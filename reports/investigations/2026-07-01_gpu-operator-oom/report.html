GPU Operator OOM Investigation

Date: 2026-07-01 | Cluster: pokprod001 | OCP: 4.15.12

Symptoms

GPU operator pods restarting every ~30 minutes with OOMKilled status.

Root Cause

Memory limit set to 256Mi but operator leaked memory during device plugin reconciliation loops.

Fix

resources:
  limits:
    memory: 512Mi
  requests:
    memory: 256Mi

Verification

After applying the resource bump, operator stable for 48+ hours with peak memory at 380Mi.
